This is a simple Angular app to test custom WP-API Routs and some AngularJS functionality.

  • Blog - displays blog posts with featured images (if set) and pagination.
  • Posts by specific Authors - displays all posts by the author (by default #1). Uses custom WP-API route.
  • Post ID's - gets all post ID's. Custom WP-API route

App files location:


Theme files

  • /lib/angular-app/
        - /templates
        - angular-app.php
        - AuthorRoute.php
        - GetIdRoute.php
    - /templates - directory contains Angular template files,
    - angular-app.php - starting point, used to include theme assets,
    - *Route.php - files used to create custom WP-API routes.

AngularJS files

  • /assets/angular-app/
    - all AngularJS app files

Page template

  • template-angular.php
    - Sage theme Page template, used as a static frontpage in WP-admin settings. This is where we start.
